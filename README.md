# dnd-witchlight-aurora
Aurora Builder homebrew pack for **The Wild Beyond the Witchlight**.

## Installation
1. Upload this repository to GitHub.
2. Copy the RAW link to `witchlight.index`, which will look like:

```
https://raw.githubusercontent.com/USERNAME/dnd-witchlight-aurora/main/witchlight.index
```

3. Open **Aurora Builder → Settings → Homebrew → Add Repository**.
4. Paste the link and click **Add**.
